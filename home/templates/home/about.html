About this project

About Us

We are a dedicated e-commerce platform focused on providing high-quality baby clothes and toys for new parents and guardians. Our mission is to make shopping for your baby's needs easy, secure, and enjoyable.

Our Features

  • Product sorting by type (clothes/toys), name, and price
  • Current promotions and discounts
  • Account registration for returning customers
  • Order tracking system
  • Smooth and secure checkout process

Technology Stack

  • HTML/CSS for frontend development
  • JavaScript for interactive features
  • Python, using Django as the backend framework
  • Docker for deployment
  • Stripe for payment processing
